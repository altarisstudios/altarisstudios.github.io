## Crafting immersive games through technology, creativity, and passion.

Welcome to the official GitHub organization of **Altaris Studio**, an independent game development studio based in Brazil.

Our goal is to create engaging, polished, and memorable gaming experiences for players around the world.

---

## Projects

### 🎮 Chromatically

A colorful puzzle game inspired by classic color sorting mechanics, featuring modern visuals, smooth animations, and an intuitive mobile experience.

Current Status:
- ✅ In development
- 📱 Android
- 🛠 Built with Godot Engine

---

### 🎖 Operation: Smoking Snakes

A historical Real-Time Strategy (RTS) game inspired by the Brazilian Expeditionary Force (FEB) during World War II.

Current Status:
- 🚧 Early Development
- 🛠 Unity 6
- 📚 Historical Research Project
