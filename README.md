## GitHub Pages

Use this repository name:

```text
altarisstudios.github.io
```

Recommended GitHub Pages settings:

- Source: Deploy from a branch
- Branch: main
- Folder: /root

The `.nojekyll` file disables Jekyll and publishes the static files directly.
